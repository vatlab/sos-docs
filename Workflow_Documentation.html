Publication¶

Please cite the following publication if you use SoS workflow engine for your work:

Script of Scripts: a simple yet powerful workflow system for daily computational research Gao Wang and Bo Peng, in preparation

Here are a list of examples from the publication:

  • Process_Oriented.sos (script)
    Demonstration of a process_oriented forward-style workflow that executes steps in pre-sepecified order.
  • Outcome_Oriented.sos (script)
    Demonstration of a outcome-oriented Makefile-style workflow that executes steps that are needed to generate specified outcome.
  • Outcome_Oriented_Step_Targets.sos (script)
    Demonstration of a outcome-oriented Makefile-style workflow that determines and executes dependencies by step targets instead of files.
  • Mixed_Style.sos (script)
    Demonstration of a mixed-style workflow that executes steps of a forward-step workflow with dependencies produced by auxiliary steps.
  • Next-generation sequencing genotyping pipeline (notebook)
    This is the SoS version of the samtools workflow: WGS/WES Mapping to Variant Calls - Version 1.0
  • RNA Seq Differential Expression Analysis Workflow (notebook and generated report)
    This is the SoS version of the RNA-seq workflow: gene-level exploratory analysis and differential expression from bioconductor.
  • RNA Seq Normalization and Expression Residuals Analysis Workflow (notebook and generated report)
    This is the SoS version of the RNA-seq preprocessing step of eQTL discovery pipeline from the GTEx Consortium , updated July 31, 2017.

Please follow instructions here to reproduce the first 5 examples.

Example Scripts and workflow notebooks¶

SoS scripts for the maintenance of SoS

  • update_toc.sos (script)
    A sos script that was used by SoS to update TOC of the wiki pages
  • release (script)
    Script to manage SoS website and release SoS to pip, docker hub, etc

Pedagogical examples

  • DEG Annotation with external task (notebook)
    The same example but submit the R part of the workflow as an external task
  • Simple External Tasks (notebook)
    An example to generate and execute a number of external tasks in Jupyter notebook. You can try to use option trunk_size to group the tasks. This example also demonstrates the use of option shared to collect results from tasks as variables
  • Remote Target (notebook)
    An example on the analysis of targets that reside on remote servers using target remote() and step option remote

Real-world applications

  • Data preprocessing for TADA-A R analysis, for Liu et al 2018, American Journal of Human Genetics.
  • Data preprocessing for MASH R analysis and Multivariate eQTL analysis pipeline, for Urbut et al 2018, Nature Genetics.
